01 관리자 토큰 확인

보안상 토큰은 화면에 표시되지 않습니다. 서버 콘솔(SSH 또는 서버 직접 접속)에서 다음 명령으로 토큰을 확인한 뒤 아래에 입력하세요.

cat {{ setup_token_path }}

서버 콘솔에서 실행 → 출력된 32자 hex 문자열을 복사 → 아래 입력란에 붙여넣기
