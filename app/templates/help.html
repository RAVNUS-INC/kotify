기능 안내

  • 새 발송 작성
  • SMS(90byte) · LMS(2000byte) · MMS(이미지 1장) 자동 판정
  • RCS 우선 → 미지원 시 SMS/MMS 자동 전환
  • 예약 발송 — 현재 시각 +10분 이후
  • 발송 이력 보기
  • 주소록 — 개별 연락처
  • CSV 일괄 가져오기
  • 그룹 — 여러 연락처 묶음
  • 그룹 내 중복 멤버는 발송 시 자동 제거
  • 발신번호 (관리자/발신자)
  • 대화방 목록
  • 고객 답장이 오면 "미답" 뱃지 표시
  • 24h 세션당 최대 10건(80원) 상한 — 반복 발신 시 자동 적용
  • 답장은 90byte 단문만 가능
  • 발송·주소록 편집: sender / admin 필요
  • 조회만 가능: viewer
  • SMS 단문 / LMS 장문 / MMS 이미지 각각 단가 상이
  • RCS 양방향 세션은 24h당 최대 10건 상한

자주 묻는 질문

발송이 실패해요

발신번호 등록 여부, 수신자 번호 형식, 메시지 길이(byte)를 먼저 확인하세요. 실패 이력에서 원인 코드를 확인할 수 있습니다.

실패 이력
예약 발송 취소

캠페인 상세에서 "예약 취소" 버튼을 누르면 됩니다. 발송 시각 5분 전까지 가능합니다.

예약 관리
대량 CSV 가져오기

CSV 가져오기에서 템플릿을 다운로드 받아 사용하세요. 중복·형식 오류는 자동 보고됩니다.

주소록
권한이 없다고 나와요

발송·주소록 편집은 sender 또는 admin 역할이 필요합니다. 내 프로필에서 현재 권한을 확인하고 관리자에게 요청하세요.
